# Project

A web application with Progressive Web App (PWA) capabilities.

## Files

- `index.html` - Main HTML file
- `manifest.json` - PWA manifest configuration
- `service-worker.js` - Service worker for offline support
- `temp_test/` - Test directory

## Getting Started

1. Clone or navigate to the project directory
2. Open `index.html` in your browser
3. Or run a local server to test PWA features

## Features

- Progressive Web App support
- Service worker for offline functionality
- Responsive design

## Setup

No external dependencies required. Simply open the files in a browser to use.
